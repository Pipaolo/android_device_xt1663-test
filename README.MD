Motorola M (XT1633)
==============

This branch is for building of the LineageOS 14.1.

# About Device

[Moto M]

# Build Information

### Working:
 * Rotation
 * Micro SD support
 * Audio
 * Auto brightness
 * Offline charging
 * Youtube
 * MTP
 * Vibration
 * All Sensors
 * LiveDisplay
 * Bluetooth
 * LED Notifications

### Not Working:
 
 * Wi-Fi tethering
 * GPS
 * AGPS
 * Flashlight (statusbar and camera)
 * Camera rear/front (video and photo) 
 * RIL for SMS/calls/internet/LTE ( Has signal but can't receive calls or make calls )
 * Wi-Fi ( Can connect but no internet connection )
 * FM Radio
 * Camera HDR
 * Hotspot (Open Hotspot)
